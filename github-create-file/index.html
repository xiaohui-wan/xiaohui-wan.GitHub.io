使用GitHub搭建自己博客

· 2020-07-03 · # GitHub

前言

最近突发奇想，想搭建一个自己的博客来记录工作以及生活中琐事。但是奈何本人忙于工作，热与生活(说白了就是懒，还是个小白)，所以一直没有空闲的时间、好的方式去搭建以及折腾这些事。毕竟要考虑服务器，域名、环境、主题，所以打算使用GitHub搭建自己博客...

开始搭建

首先，注册GitHub账号

  • 输入用户名（Username）
  • 输入邮箱（Email）
  • 输入密码（Password）
  • 去邮箱验证

登陆（已经登陆的话可以忽略）

  • 示例图

创建存储库

  • 点击右上角的+，选择New repository.
  • 示例图
  • 输入存储库名称（Repository name）
  • 示例图
  • 点击按钮Creating repository进行创建。
  • 创建成功后，点击Settings进入设置,去设置主题样式.
  • 示例图
  • 进入设置页面后，下拉找到GitHub Pages点击Choose a theme去设置主题
  • 示例图
  • 进入后可以看到GitHub提供了不少的主题(应该有十来个吧)，找一个喜欢的，点击Select theme选择即可。
  • 示例图
  • 选择主题后GitHub Pages 就会生成网站，输入你刚才设置的地址，比如我的xiaohui-wan.GitHub.io就可以打开了。（https://xiaohui-wan.github.io/）

Wan（小提示）

  • 创建存储库名称时建议使用:你的用户名称+.GitHub.io,比如我的存储库名称xiaohui-wan.GitHub.io,在地址栏可以直接输入进行访问

  • 准备设置主题样式时注意下绿色框内的信息，这是你的访问地址，如果忘记或者访问失败可以进入此页面查看此地址

  • GitHub提供的主题比较丑，当然也可以自己去折腾一些好看的主题
